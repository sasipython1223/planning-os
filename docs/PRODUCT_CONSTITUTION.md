# Product Constitution

## 10–20 Year Vision
Planner-Studio is not a UI tool for drawing Gantt charts. It is a planning, scheduling, and impact analysis system for complex projects, designed for long-term durability, auditability, and progressive refinement.

## Core Tenets

1. **The Project Model is the Source of Truth**  
   The schedule is a computed projection of the underlying project model.

2. **Narrative is Structured Data**  
   Assumptions, constraints, methods, and decisions that affect computation must be stored as structured, queryable data, not hidden in free-text fields or external documents.

3. **Auditability is Non-Negotiable**  
   Every meaningful state change must be traceable in principle. The system must always move toward being able to answer what changed, when it changed, and why it changed.

4. **Progressive Refinement is Native**  
   The platform must support movement from low-fidelity assumptions to high-fidelity execution detail without breaking the model.

5. **AI Operates Under Governance**  
   AI is treated as a governed actor. It must not have backdoor access to canonical state and must enter through the same command and validation pathways as other system actions.

6. **Branching and Simulation are First-Class Future Capabilities**  
   The platform must be architected to support parallel scenarios, what-if analysis, and comparison against a canonical baseline.
